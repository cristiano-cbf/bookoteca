O projeto é um sistema de catálogo de livros, que permite aos usuários
pesquisar, visualizar e navegar por uma lista de livros cadastrados. 
Com uma interface intuitiva, os usuários podem facilmente acessar detalhes
sobre cada obra.
Desenvolvido em PHP e utilizando MySQL para gerenciamento de dados, o 
projeto foi criado com o foco em proporcionar uma experiência 
fluida e agradável aos usuários que desejam explorar o acervo da plataforma.
